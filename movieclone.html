Netflix Clone

front-end dev

This project was part of the Scrimba Front-End Career Path. The goal of this project was to create a netflix streaming clone with the help of instructor Karl Hadwen.

This project was built using React, Styled Components, and used Firebase for authentication.

Some challenges that I faced were: Getting my authentication to work, and getting the styling to properly link with the nested components. Make sure to make a new account to access the main page of the site!
